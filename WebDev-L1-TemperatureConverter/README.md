# Temperature Converter

## Description
A responsive temperature converter website that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Features
- Temperature Input
- Unit Selection
- Convert Button
- Result Display
- Input Validation
- Absolute Zero Validation
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
